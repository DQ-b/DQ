# LittleSoldier3D

Unity first-stage FPS prototype generated from `Codex任务书-FPS第一阶段.md`.

## Run

1. Open this folder with Unity 6.
2. Confirm `Edit > Project Settings > Player > Other Settings > Active Input Handling` is set to `Both` or `Input Manager (Old)`.
3. Open any scene, then run `LittleSoldier > Build Training Scene`.
4. Press Play.

## Test

- WASD moves, Shift runs, Space jumps.
- Mouse moves the first-person camera.
- Left click fires a raycast shot.
- Targets disappear after enough hits.
- Escape unlocks the cursor.
